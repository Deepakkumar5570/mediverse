# Purpose

Defines every learning resource attached to a Topic.

Topic is the primary learning object.

Every educational resource belongs to exactly one Topic.

Examples

- Explain the structure of a cell membrane.
- Differentiate passive and active transport.
- Identify membrane proteins.

# Design Principles

1. Topic owns every learning resource.

2. Resources can evolve independently.

3. Resources are reusable.

4. Every resource is searchable.

5. Every resource is versioned.

6. Every resource supports analytics.
